makemovies

PURPOSE

[4D] Used to convert R sets of equal length videos into a single movie.

SYNOPSIS

function M = makemovies( IS, montage2params )

DESCRIPTION 

 [4D] Used to convert R sets of equal length videos into a single movie.

 To display same data statically use montages.
 Works by calling montage2 once per frame.

 USAGE
  M = makemovies( IS, [montage2params] )

 INPUTS
  IS              - MxNxTxR or MxNx1xTxR or MxNx3xTxR array, or cell array
                    where each element is a MxNxT or MxNx1xT or MxNx3xT
  montage2params  - [] params for montage2, ex: {showlines, extrainfo}

 OUTPUTS
  M               - resulting movie

 EXAMPLE
  load( 'images.mat' );
  M = makemovies( videos );
  movie( M );

 See also MONTAGES, MONTAGE2, MAKEMOVIE, MAKEMOVIESETS
